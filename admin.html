📖 トークンの取得方法

  1. GitHub Fine-grained tokens を開く
  2. 「Generate new token」をクリック
  3. Token name: prohub-admin など任意の名前
  4. Repository access: 「Only select repositories」→ FPS を選択
  5. Permissions → Repository permissions → Contents: Read and write
  6. 「Generate token」で作成し、トークンをコピー
